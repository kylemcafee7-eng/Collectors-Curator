# Collectors-Curator

Collectors Curator is a C# WinForms application backed by a MySQL database that helps users organize and analyze personal collections by providing structured item storage, detailed attributes, and fast searching tools. It allows collectors to add, edit, and categorize items with fields like name, condition, quantity, value, and notes, while offering filters, sorting, and keyword search for navigating large collections. The system includes predictive logic to highlight missing items, duplicates, and value trends, giving users insight into the completeness and worth of their collection. Built with a relational database and a clean, user‑friendly interface, the project demonstrates practical skills in C#, SQL, CRUD operations, UI design, and real world problem solving.
